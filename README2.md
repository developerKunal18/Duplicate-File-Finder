# Duplicate File Finder

A Python script that scans a folder and detects duplicate files based on file size.

## Description
The program checks all files in a specified folder and compares their file sizes.
If two files share the same size, they are considered potential duplicates.

## Features
- Folder scanning
- Duplicate detection
- Simple and beginner-friendly logic
- Useful for file organization

## Technologies Used
- Python
- os module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the program:

   python duplicate_finder.py

4. Enter the folder path when prompted
